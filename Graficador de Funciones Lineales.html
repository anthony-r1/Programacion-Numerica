Graficador de Funciones Lineales

Visualización en Plano Cartesiano ASCII

📖 Descripción del Proyecto

Una función representa una relación matemática entre dos conjuntos, donde cada elemento del dominio (entrada) se asocia con exactamente un elemento del codominio (salida). Comúnmente se expresa mediante la notación:

y = f(x)

Las funciones lineales son aquellas cuya representación gráfica forma una línea recta en el plano cartesiano. Su expresión algebraica general es:

f(x) = m·x + b
📐 m (pendiente)
Indica la inclinación de la recta. Determina qué tan empinada o plana es la línea.
• m > 0: la recta sube de izquierda a derecha
• m < 0: la recta baja de izquierda a derecha
• m = 0: la recta es horizontal
📍 b (ordenada al origen)
Representa el punto donde la recta intersecta el eje Y. Es el valor de la función cuando x = 0.

🎯 Objetivo del Graficador

Este programa desarrollado en Python tiene como propósito reconocer y visualizar dos funciones lineales en un plano cartesiano utilizando caracteres ASCII.

La implementación está diseñada con las siguientes características:

🖥️
Interfaz de Consola
Interfaz simple y accesible que funciona directamente en la terminal, sin necesidad de librerías gráficas externas.
📏
Plano Limitado
El rango de graficación se limita a 20 unidades por coordenada (-15 a +15 en ambos ejes) para una visualización óptima.
🎨
Sin Librerías Gráficas
Implementación pura en Python sin dependencias de librerías especializadas como matplotlib o numpy.
🔍
Identificación Visual
Sistema de símbolos distintivos para cada función y sus intersecciones, facilitando la interpretación del gráfico.

💻 Código del Graficador

ℹ️ Funcionamiento del Programa

El programa solicita al usuario dos funciones lineales en formato algebraico (por ejemplo: "2x+1" o "-x+3"), analiza sus componentes (pendiente y ordenada), y luego genera una representación visual en el plano cartesiano usando caracteres ASCII.

🐍 Python 
def graficar_lineas_ascii():
    try:
        # Solicitar las funciones al usuario
        ecuacion1 = input("Ingrese la primera función (ejemplo: 2x+1): ").replace(" ", "")
        ecuacion2 = input("Ingrese la segunda función (ejemplo: -x+3): ").replace(" ", "")

        def get_mb(eq):
            # Extraer pendiente (m) y ordenada (b) de la ecuación
            if 'x' not in eq:
                return 0, float(eq)
            
            # Calcular m (pendiente)
            m_str = eq.split('x')[0]
            m = 1 if m_str in ('', '+') else -1 if m_str == '-' else float(m_str)
            
            # Calcular b (ordenada al origen)
            b_str = eq.split('x')[1] if len(eq.split('x')) > 1 else ''
            b = float(b_str) if b_str else 0
            
            return m, b

        # Obtener parámetros de ambas funciones
        m1, b1 = get_mb(ecuacion1)
        m2, b2 = get_mb(ecuacion2)

        # Mostrar leyenda de símbolos
        print("\nLeyenda:\n* = F1\no = F2\n# = Intersección\n| = Eje Y\n- = Eje X\n+ = Origen")
        
        # Generar el plano cartesiano
        for y in range(15, -16, -1):
            linea = ""
            for x in range(-15, 16):
                # Verificar si el punto pertenece a cada función
                on_f1 = round(m1 * x + b1) == y
                on_f2 = round(m2 * x + b2) == y
                
                # Asignar símbolo según la posición
                if on_f1 and on_f2:
                    linea += "#"  # Intersección
                elif on_f1:
                    linea += "*"  # Función 1
                elif on_f2:
                    linea += "o"  # Función 2
                elif x == 0 and y == 0:
                    linea += "+"  # Origen
                elif x == 0:
                    linea += "|"  # Eje Y
                elif y == 0:
                    linea += "-"  # Eje X
                else:
                    linea += " "  # Espacio vacío
            print(linea)

    except (ValueError, IndexError):
        print("\nError: La ecuación ingresada no es válida.")

# Ejecutar el programa
graficar_lineas_ascii()

  • - = Eje X (horizontal)

💡 Análisis del Resultado

En este ejemplo, podemos observar que las funciones f₁(x) = 2x+1 y f₂(x) = -x+3 se intersectan en el punto marcado con "#". La función f₁ tiene pendiente positiva (sube), mientras que f₂ tiene pendiente negativa (baja).
